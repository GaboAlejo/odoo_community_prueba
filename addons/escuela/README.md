# Módulo Escuela

## Descripción
Este módulo permite gestionar la información de los alumnos de una institución educativa.

## Características
- Registro completo de alumnos con información personal
- Gestión de datos académicos (grado, fecha de ingreso)
- Información de contacto y emergencia
- Números de matrícula automáticos
- Vistas de formulario, lista, kanban y búsqueda
- Sistema de permisos y seguridad

## Campos del Alumno
- **Información Personal**: Nombre, apellidos, fecha de nacimiento, edad
- **Información Académica**: Grado, número de matrícula, fecha de ingreso
- **Contacto**: Email, teléfono, dirección
- **Emergencia**: Contacto de emergencia, teléfono, parentesco
- **Adicional**: Notas, estado activo/inactivo

## Instalación
1. Copiar el módulo en el directorio addons de Odoo
2. Actualizar la lista de aplicaciones
3. Instalar el módulo "Escuela"

## Uso
Una vez instalado, podrás acceder al módulo desde el menú principal "Escuela" > "Alumnos" para gestionar los estudiantes.
